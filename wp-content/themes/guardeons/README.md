# GuardEons WordPress Theme

Modern, professional theme for GuardEons IT services & cybersecurity.

## Features
- Custom Post Types: Services, Case Studies, Team Members, Job Openings
- Taxonomies: Industry (Case Studies), Service Category (Services)
- Front page sections: Hero, Services, About, Case Studies, Team, Careers, Contact
- Customizer: Colors, Typography, Contact info
- Menus: Primary, Footer, Social
- Widget areas: Header Top, Sidebar, Footer (3)
- Responsive, accessibility-minded, SEO-friendly

## Installation
1. Copy the `guardeons` folder into `wp-content/themes/`.
2. In WordPress admin, activate the GuardEons theme.
3. Create menus for Primary, Footer, Social and assign in Appearance > Menus.
4. Add content to the Custom Post Types.
5. Customize colors/typography in Appearance > Customize.

## Recommended Plugins
- Advanced Custom Fields (optional) for richer fields
- WPForms or Contact Form 7 (optional) if you prefer plugin forms
- Regenerate Thumbnails
- Caching plugin (e.g., WP Super Cache or W3 Total Cache)
- SEO plugin (e.g., Yoast SEO or Rank Math)

## Development
- Styles in `assets/css/main.css`
- Scripts in `assets/js/main.js`
- Section templates in `template-parts/components/`
